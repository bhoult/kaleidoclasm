# Controls

## Selection and Movement

| Input | Action |
|-------|--------|
| Left Click on unit | Select unit |
| Left Click on tile | Move selected unit (if in range) |
| Left Click on enemy | Attack enemy (if unit selected and adjacent) |

## Context Menu

| Input | Action |
|-------|--------|
| Right Click on tile/object | Open context menu with available actions |

Actions require an adjacent unit and cost AP.

## Cards

| Input | Action |
|-------|--------|
| Click card | Play card on selected unit |

## Camera

| Input | Action |
|-------|--------|
| Mouse drag | Rotate camera (constrained above ground) |
| Scroll wheel | Zoom camera |

## UI Buttons

| Button | Action |
|--------|--------|
| End Turn | Advance to next turn |
| Save | Persist game to localStorage |
| Load | Restore game from localStorage |

## Tooltip

Hover over tiles to see:
- Terrain type
- Objects on tile
- Radiation level
- Movement cost
- Passability
